SECRET MANIFEST - do not commit real values

本檔只列出需要設定的 Secret 名稱，不包含真實 API Key。
真實 Key 請只放在 Supabase Secrets / Cloudflare Worker Secrets / GitHub Actions Secrets。

Supabase Edge Function Secrets:
- ANTHROPIC_API_KEY：Claude 文案生成
- CARTESIA_API_KEY：語音克隆 / TTS
- RUNNINGHUB_API_KEY：形象克隆任務
- SHOTSTACK_SANDBOX_API_KEY：測試頁字幕燒錄成片
- SUPABASE_URL：Supabase Project URL
- SUPABASE_SERVICE_ROLE_KEY：後端內部資料庫操作，禁止放前端
- AVATAR_R2_WORKER_URL：Cloudflare R2 Worker URL
- AVATAR_R2_WORKER_SECRET：Supabase 呼叫 R2 Worker 的內部密鑰
- CHIWA_AI_BACKEND_SECRET：前端不可見的內部後端密鑰，依實際 Worker 使用情況設定

Cloudflare Worker Secrets:
- AVATAR_R2_WORKER_SECRET：限制 Supabase 後端才可匯入/簽署影片
- RUNNINGHUB_API_KEY：若 Worker 端有直接代理形象任務才需要
- SUPABASE_URL：如 Worker 需要查詢 Supabase 才設定
- SUPABASE_SERVICE_ROLE_KEY：如 Worker 需要後端權限才設定

Public frontend config allowed in HTML:
- Supabase anon public key 可以存在前端，但 service_role key 絕對不可放前端。

禁止事項：
- 不要把 Cartesia、RunningHub、Shotstack、Supabase service_role 等真實 API Key 寫進 index.html。
- 不要把真實 API Key 放進 GitHub Pages。
- 不要把 API Key 放進 localStorage 或前端可讀變數。
